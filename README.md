# ED-dashboard

A simple React and TypeScript program for managing hospital bed assignments. The main component `Lov≈≥ValdymoPrograma.tsx` contains the dashboard logic.

## Setup

1. Install [Node.js](https://nodejs.org/) and `npm`.
2. Install dependencies:
   ```sh
   npm install
   ```
3. Run the tests to ensure everything works:
  ```sh
  npm test
  ```

4. Build the static files for GitHub Pages (output in the `docs/` folder):
   ```sh
   npm run build
   ```
